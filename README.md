# IndieMed

## Overview
IndieMed is a healthcare web platform built to help people in rural areas find quick and affordable medical support online.  
The project focuses on connecting patients with certified doctors, allowing users to book appointments and filter consultations based on budget and urgency.

## Features
- Online doctor appointment booking
- Doctor filtering based on patient budget
- Consultation sorting based on urgency
- Healthcare access for rural and remote users
- Responsive design for mobile and desktop
- Simple and user-friendly interface
- Doctor listing with key details
- Easy navigation for patients seeking quick medical help

## Tech Stack
- HTML
- CSS
- JavaScript
- Backend basics
- Database integration
- Responsive Web Design
- InfinityFree Hosting
- Git & GitHub
